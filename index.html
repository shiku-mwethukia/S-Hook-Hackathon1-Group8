Projects

For the 3 years i have completed in campus I have done two school projects.

  1. Handwritten Signature Verification system

    2. In my 2nd year of study, in a group of 7 we created a handwritten verification system which would verify students' signature in Kenyan universities. The system was written in python and it incorporated machine learning specifically AlexNet architecture.

    Despite all our efforts we did not deploy the system because it was not accurate. This was due to the computationally expensive training model and lack of specific hardware requirements to train the system.

  2. Lugha Link Virtual Translator

    3. In 3rd year, in a group of 4 we created an English to Swahili traslation virtual assistant which would be used by swahili learners to translate English words and phrases to Swahili. The system was written in python and used tensorflow and kaggle datasets for training to translate between English and Swahili. NLTK library for Python was used to train the model to identify and process natural languages.
